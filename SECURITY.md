## Reporting a Vulnerability

We take the security of CelestialMechanics seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### Process

1.  E-mail your findings to `security@celestialmechanics.example.com`.
2.  Do not disclose the vulnerability publicly until we have had a chance to fix it.
3.  We will respond to your report within 48 hours.

### What we do

*   We will confirm the problem and determine a plan for a fix.
*   We will communicate with you progress and set a release date for the fix.
*   We will credit you for your discovery (unless you prefer to remain anonymous).

### Non-vulnerabilities

The following are not considered security vulnerabilities:

*   Denial of Service (DoS) attacks.
*   Vulnerabilities visible only to the server operator/owner.

Thank you for helping keep CelestialMechanics safe!
